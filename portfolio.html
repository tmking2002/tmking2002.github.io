Softball Statline is a website that provides for easy access to historical archived NCAA softball statistics. What started as a passion project to help grow the game has turned into a one-stop shop for casual softball fans, diehards, and coaches alike to access the statistics of their favorite players.

I scraped this data over the past year from the NCAA stats archive using a scraper I built in R. I built the website using just HTML, CSS, and JS with the help of Jekyll.

Check out the link above to check out the site for yourself!

softballR is an R package I built with functionality to scrape, organize and dynamically load softball data and was designed to be user-friendly for a newcomer to programming. The package allows easy access to historical scores, player stats, rankings, and much more.

Click the links above to see the GitHub repository and a tutorial that I wrote to help newbies get started.

Call Center Text Classification Project

In January 2024, I worked closely with two classmates to compete in a Best Buy sponsored project to create a classification model for call center interactions.

Our final workflow consisted of text lemmatization, vectorization, PCA dimension reduction and then a multilayer perceptron neural network. We won this competition and won $600! Check out the link above to see our presentation.
